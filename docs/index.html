    Configuration

    variables

    base

    $base: () !default;

    Description

    The grid is defined in a single map variable, with four initial properties: font-size, row-height. Anything you put in the root $base variable map will be treated as a global project default. You can create similar configuration maps under different variable names, to override the defaults as-needed.

    Type

    Map

    Map structure

    Map key NameMap key DescriptionMap key TypeMap key Value
    root-font-size

    Font-size of the root element

    		Number— none
    row-height

    Height of each row, generaly 1/2 of the font-size

    		Number.5
    min-line-height

    Minimum line-height

    		Number1.2
    cap-height

    Cap-height of the used font. Possible resource https://jamonserrano.github.io/plumber-sass/baserows/

    		Number— none

    Example

    Default values

    $base: (
  font-size: 1rem,
  row-height: .5
);

    Example with Segoe UI as font

    $base: (
  font-size: 1rem,
  row-height: .5,
  cap-height: 0.86
);

    Used by

    • [function] _base-settings

    @function root-font-size($short, $config: $base) { ... }

    Description

    It returns the extracted font-size from the shorthand, if non given it will fallback on the base font-size.

    The un-prefixed alias font-size() is available by default.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $short

    Shorthand expression to define the current text properties, optionally containing:

    • a unit number, current font-size
    • a unitless number, extra rows to add per text line
    		List— none
    $config

    Optional map of Base configuration settings. See $base documentation for details.

    		Map$base

    Returns

    Length —

    Raw font-size

    Example

    .foo {
  font-size: root-font-size(1em);
}
